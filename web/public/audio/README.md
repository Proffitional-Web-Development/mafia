## Audio files

This directory should contain the `game-ambient.mp3` file for background music.
Phase cues should be placed under `audio/phases/`.

You can download a suitable ambient track from any royalty-free source (e.g. Pixabay) and place it here.
The frontend expects a file named `game-ambient.mp3`.

Required phase files:

- `audio/phases/discussion.mp3`
- `audio/phases/publicVoting.mp3`
- `audio/phases/abilityPhase.mp3`
- `audio/phases/mafiaVoting.mp3`
- `audio/phases/resolution.mp3`
- `audio/phases/roundEnd.mp3`

Please fill `audio/CREDITS.md` with source URL, license, and author for each asset.
